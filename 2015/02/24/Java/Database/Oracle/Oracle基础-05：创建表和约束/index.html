约束

Oracle有以下五种约束：

  • NOT NULL：非空。
  • UNIQUE：唯一。
  • PRIMARY KEY：主键。
  • FOREIGN KEY：外键。
  • CHECK：校验，例如性别只有男和女，年龄大于等于0等这些情况。

定义约束时如果不指定约束名 ，Oracle 自动按照 SYS_Cn 的格式指定约束名。

基本语法：
