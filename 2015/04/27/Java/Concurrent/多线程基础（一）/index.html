概述

进程是一个正在执行的程序。
线程是进程中一个独立的控制单元，线程控制者进程的执行。
Java虚拟机允许应用程序并发地运行多个执行线程。

基础知识

1）主线程和其它线程

JVM的主线程调用main方法，main方法中可以创建新的执行路径，即新的线程。

2）run与start方法的区别

若创建一个线程后执行的是run方法，则这个线程并没有开启，只是执行run方法里面的代码，程序是单线程的。
只有执行的是start方法，才会开启线程并自动执行run 方法。

3)Thread 相关方法介绍

每个线程都有一个标识名，多个线程可以同名。如果线程创建时没有指定标识名，就会为其生成一个新名称。命名规则：Thread-编号，编号从0开始。
还能使用getName和setName方法获取和设置名称。
currentThread获取当前的线程对象，是Thread的静态方法，能直接使用。
